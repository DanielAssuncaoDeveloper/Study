Algoritmos
==================

- Uma sequencia de instruções, que atingem um determinado objetivo
// Exemplo da "receita de bolo"

- Uma instrução para a máquina deve ser precisa, ao contrario de uma conversa humana onde, o receptor consegue interpretar a comunicação. Para realizar essa comunicação com a máquina, é necessário uma linguagem específica, de certa forma "lógica", onde a precição da comunicação é o que define uma linguagem de programação.

Conceitos que formam uma linguagem de prog
=============
1. Comando
// Um termo que passado para o computador fazer algo

2. Parâmetros
// Um valor que é informado aos comandos para que eles possam realizar suas ações

3. Sintaxe
// As regras dos comandos
// Para o computador, a sintaxe tem que ser precisa.


5 Passos para elaborar um algoritmo
===================================
1. Entender o problema. Entender o que o algoritmos tem que resolver
2. VOCÊ tem que saber como resolver o problema
3. Criar uma sequencia de passos para resolver o problema
4. Implementar o algoritmo
5. Validação da sua implementação
